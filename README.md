# Event Management API

### Technologies
- Nest.js
- Typescript
- SQLite3
- TypeORM
- Swagger
### Install all packages using NPM
```bash
npm i
```
### Run Migrations 
```bash
npm run typeorm:run-migrations
```
### Usage
```bash
npm run start:dev
```
## Test
```bash
npm run test
```
## Access Swagger UI
```bash
localhost:3000/api
```
